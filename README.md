# Bean Box – Coffee Shop Website

Bean Box is a responsive coffee shop website designed using HTML, CSS, and JavaScript.
It showcases handcrafted coffee beverages, store locations, and provides an interactive and modern user experience.

## Features

- Responsive design (Desktop & Mobile friendly)
- Fixed navigation bar with smooth scrolling
- Animated homepage section
- Coffee product showcase with images
- Branches table with Sri Lankan locations
- Interactive search bar toggle
- Styled footer with social media links
- Clean and organized UI design

## Technologies Used

- HTML5 – Page structure
- CSS3 – Styling, animations, responsive design
- JavaScript – Interactivity and functionality

## How to Run the Project

- Download or clone the repository:
```
git clone https://github.com/kivindisam/bean-box.git
```
- Open the project folder.
- Double-click index.html to open in your browser.

No additional installations are required.
